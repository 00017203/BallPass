# BallPass – Football Ticket Exchange Platform

## Project Overview
BallPass is a web-based platform designed to provide a secure and transparent environment for football fans to buy, sell, and exchange tickets. The system focuses on reducing fraud and improving trust in the ticket resale market.

This project was developed as part of the BSc (Hons) Business Information Systems degree.

## Key Features
- User registration and login system
- Ticket listing and browsing
- Simulated ticket purchasing
- User profiles and transaction history
- Trust indicators (verification status)

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Storage: LocalStorage (MVP stage)

## How to Run the Project

1. Clone the repository:
```
git clone https://github.com/YOUR-USERNAME/BallPass.git
```

2. Navigate to the project folder:
```
cd BallPass
```

3. Install dependencies:
```
npm install
```

4. Run the server:
```
node app.js
```

5. Open in browser:
```
http://localhost:3000
```
